# 📋 Changelog

## [1.1.0] - 2025-03-11

### ✨ 새 기능

- **자막 내보내기** — 번역된 자막을 `.md` / `.txt` 형식으로 다운로드 가능
- **언어 옵션 선택** — 내보내기 시 영한 / 영어 중 선택 가능
- **파일명 자동 설정** — 현재 브라우저 탭 제목 기반으로 파일명 자동 생성
- **파일명 언어 구분 표기** — 선택한 언어에 따라 `_영한자막` / `_영어자막` suffix 추가
- **파일명 중복 방지** — 동일한 이름 다운로드 시 `_(1)`, `_(2)` 자동 추가

### ⚡ 성능 개선

- **번역 속도 개선** — `translateCues` 배치 내 순차 처리 → `Promise.all` 병렬 처리로 변경 (최대 10배 향상)

### 🐛 버그 수정

- **VTT 파싱 오류 수정** — cue 사이 빈 줄이 없는 포맷에서 타임스탬프가 텍스트에 섞이는 문제 수정
  - 기존 블록(`\n\n`) 기반 파싱 → `-->` 라인 기준 파싱으로 전면 교체
  - cue 번호, 헤더, NOTE/STYLE/REGION 블록 등 불필요한 줄 필터링 추가
- **Service Worker 슬립 이슈 수정** — `lastTranslatedCues` 메모리 변수 → `chrome.storage.local` 저장 방식으로 변경하여 SW 재기동 시 데이터 유실 방지

### 🔧 기타

- `manifest.json` — `tabs` 권한 추가 (탭 제목 읽기용)
